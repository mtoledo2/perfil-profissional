# Murilo Toledo - Página Profissional

Este projeto é uma página profissional desenvolvida para apresentar minhas informações de contato, formação e experiência na área de Tecnologia da Informação.

## Tecnologias Utilizadas

- React
- Vite
- Tailwind CSS
- React Icons

## Sobre o Projeto

A aplicação foi desenvolvida com foco em simplicidade, responsividade e boa experiência do usuário.  
O objetivo é servir como cartão profissional online para compartilhamento em processos seletivos e networking.
